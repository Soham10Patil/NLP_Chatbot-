# 🤖 AI Chatbot using Hugging Face Transformers

This project is a conversational chatbot built using a pre-trained transformer model (DialoGPT) from Hugging Face.

## 🚀 Features
- Interactive chatbot (console-based)
- Context-aware conversation
- Dynamic response generation
- Exit handling (type 'exit')

## 🧠 Tech Stack
- Python
- Hugging Face Transformers
- PyTorch

## 🔄 Pipeline
User Input → Tokenization → Transformer Model → Response Generation → Output
